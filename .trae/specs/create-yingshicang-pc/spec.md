# 影视仓 PC版 Spec

## Why
目前市面上流行的“影视仓”（基于 TVBox 开源项目衍生的聚合影视应用）仅有 Android 手机版和 TV 盒子版，缺乏原生的 PC 桌面版本。用户在电脑端观影时体验不佳。开发一个支持导入自定义数据源（JSON 接口）、纯净无广告的 PC 版影视仓，能填补这一空白，大幅提升 PC 用户的观影体验。

## What Changes
- 初始化基于 Electron/Tauri + React/Vue 的桌面端项目。
- 开发“数据源管理”功能：支持输入 TVBox 格式的 JSON 数据源地址，解析并存储到本地。
- 开发“首页展示”功能：解析数据源后，展示推荐的影视资源、分类列表。
- 开发“搜索功能”：支持在当前数据源中搜索影视剧。
- 开发“详情与选集”功能：展示影视详情、线路切换、剧集列表。
- 开发“视频播放器”：集成支持 m3u8、mp4 等主流流媒体格式的播放器（如 DPlayer 或 Video.js）。

## Impact
- Affected specs: 影视数据解析模块、本地存储模块、视频播放模块。
- Affected code: 这是一个从零开始的全新项目，包含完整的前端页面与桌面端壳子。

## ADDED Requirements
### Requirement: 数据源解析与管理
系统应允许用户输入和切换不同的 TVBox 格式 JSON 配置链接，并解析其中的分类、首页数据及搜索接口。
#### Scenario: 成功导入数据源
- **WHEN** 用户在设置中输入合法的 JSON 数据源链接并点击保存
- **THEN** 系统解析链接，提示“导入成功”，并自动刷新首页数据，展示接口提供的影视分类。

### Requirement: 影视内容搜索与播放
系统应能搜索选定的影视剧集，并使用内置播放器进行流畅播放。
#### Scenario: 成功搜索并播放视频
- **WHEN** 用户在搜索框输入关键词或在详情页点击某一集
- **THEN** 系统请求真实的流媒体播放地址，并在视频播放器中开始播放。
